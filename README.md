# Math 2050 - Applied Statistics with Programming

This course provides an introduction to statistical programming from describing raw data (descriptive statistics) to making statistical conclusions (inferential statistics) based on real data from practical problems.

## About this repository

This repository contains course materials for Math 2050, including Python scripts, sample data sets, and homework practice files. The focus is on using programming to explore, analyze, and visualize real-world datasets as part of statistical learning.

## Contents

- `*.py` files: Python scripts for practice problems, homework assignments, and project work.
- `*.csv` files: Data sets used for analysis and examples, such as `AAPL.csv`, `heart.csv`, and `kc_house_data.csv`.
- `README.md`: This overview and documentation file.

## How to use this repository

1. Open a Python editor or notebook environment (for example, VS Code, Jupyter Notebook, or Google Colab).
2. Load relevant CSV files using pandas or another data analysis library.
3. Run or adapt the Python scripts to reproduce analyses, practice problems, and visualization examples.
4. Save findings, interpretations, and conclusions as part of homework or project deliverables.

## Recommended workflow

- Start with the practice files to review basic techniques such as summary statistics, probability distributions, and regression.
- Use the dataset files to test code and explore relationships between variables.
- Refer to the script names for topic guidance, such as `Linear Regression Practice.py`, `Logistic Regression Practice.py`, or `Confidence Intervals Practice.py`.

## Course learning outcomes

At the successful conclusion of this course, students will be able to:
- Interpret, summarize, and graph practical data sets using statistical programming.
- Conduct appropriate statistical analysis of practical data sets using statistical programming.
- Interpret and understand the results of statistical analyses from statistical programs.
- Develop conclusions and decisions based on statistical analysis results.
