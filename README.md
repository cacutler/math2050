# Math 2050 - Applied Statistics with Programming
This course provides an introduction to statistical programming from describing raw data (descriptive statistics) to making statistical conclusions (inferential statistics) based on real data from practical problems.
# Course Learning Outcomes
At the successful conclusion of this course, students will be able to:
- Interpret, summarize, and graph practical data set using statistical programming.
- Conduct appropriate statistical analysis of practical data set using statistical programming. 
- Interpret and understand the results of statistical analyses from statistical program.
- Develop conclusions and decisions based on the statistical analysis results.
